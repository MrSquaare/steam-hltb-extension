## About

Steam HLTB is a browser extension for Chrome and Firefox that adds HowLongToBeat times on Steam game store pages.

### Disclaimer

This project is not affiliated with or endorsed by HowLongToBeat or Steam.

### Built with

- [React](https://react.dev/)
- [WXT](https://wxt.dev/)

## Using

Install from Chrome Web Store or Firefox Add-ons, or build from source and load as an unpacked extension.
