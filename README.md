## About

Steam HLTB is a browser extension for Chrome/Edge/Opera and Firefox that adds HowLongToBeat times on Steam game store pages.

### Disclaimer

This project is not affiliated with or endorsed by HowLongToBeat or Steam.

### Built with

- [React](https://react.dev/)
- [WXT](https://wxt.dev/)

## Using

### Install from store

- For Chrome/Edge/Opera: [Chrome Web Store](https://chromewebstore.google.com/detail/cnfpnjgnkmonaeincdllnfeopmnngiin)
- For Firefox: [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/steam_hltb/)

### Install manually

- Download from [latest release](https://github.com/MrSquaare/steam-hltb-extension/releases/latest) or [bundle from source](CONTRIBUTING.md#bundling)
- For Chrome:
  - Uncompress the zip file
  - Follow [these instructions](https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world#load-unpacked) (only read the "Load an unpacked extension" section)
- For Firefox:
  - Follow [these instructions](https://extensionworkshop.com/documentation/publish/install-self-distributed/#install-addon-from-file) (only read the "Install add-on from file" section)
